首先在对应位置插入所需文献的交叉引用，例如[1][2]。

选中插入的引用，右键选择 切换域代码。

得到例如：

```
{ REF _Ref227822518 \r \h }{ REF _Ref227822519 \r \h }
```

如果需要去除 [1] 的“]”，则在代码中添加“\#"[0"”，如果需要去除“[”，则在代码中添加“\#"0]"”。

例如：

```
{ REF _Ref227822518 \r \h \#"[0" }{ REF _Ref227822519 \r \h \#"0]" }
```

重新切换域代码并更新域后，则会得到[12]。

手动插入“-”，如：

```
{ REF _Ref227822518 \r \h \#"[0" }-{ REF _Ref227822519 \r \h \#"0]" }
```

重新切换域代码并更新域后，则会得到[1-2]。

如果超过则2个引用，例如[1][2][5]，非两端的引用则需要在代码中插入“\#"0"”，例如：

```
{ REF _Ref227822518 \r \h \#"[0" }{ REF _Ref227822519 \r \h \#"0" }{ REF _Ref227822522 \r \h \#"0]" }
```

重新切换域代码并更新域后，则会得到[125]。

手动插入“-”、“,”，如：

```
{ REF _Ref227822518 \r \h \#"[0" }-{ REF _Ref227822519 \r \h \#"0" },{ REF _Ref227822522 \r \h \#"0]" }
```

重新切换域代码并更新域后，则会得到[1-2,5]。
