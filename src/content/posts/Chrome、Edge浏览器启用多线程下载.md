Chrome浏览器，地址栏输入：

```
chrome://flags/#enable-parallel-downloading
```

Edge浏览器，地址栏输入：

```
edge://flags/#enable-parallel-downloading
```

将默认的***Default***选项改为***Enabled***后重启浏览器即可。
